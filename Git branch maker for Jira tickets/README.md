# Branch Maker Bookmarklet

Creates Github/gitlab branches from Jira tickets.

## Usage

1. Add the bookmarklet to your browser and paste the JS function in the url
2. Browse a Jira ticket (e.g. `ITDEV-1256`)
3. Click the bookmark
4. Enter your Git project path (saved and autocompleted for next time)
5. Choose branch type like `feature`,`hotfix` etc. (saved and autocompleted for next time)
6. You will be taken directly to the git branch creation page with the name filled in like `feature/ITDEV-1256`

* You need to be logged in git but not api token needed
